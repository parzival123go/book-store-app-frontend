# 📚 MERN Stack Bookstore – Full-Stack E-Commerce Application

A complete **full-stack bookstore web application** built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js). This project includes user authentication, cart management, checkout flow, order processing, and a secure admin dashboard for managing books and inventory.

---

## 🔗 Project Links

**Live Demo:**
👉 https://book-store-app-frontend-ten.vercel.app/

---

## 🚀 Project Overview

This is a production-style **MERN stack bookstore platform** featuring:

* User authentication (JWT-based)
* Shopping cart functionality
* Order creation and management
* Admin dashboard with protected routes
* Full inventory management system

The application demonstrates real-world full-stack architecture with a clean separation between frontend and backend, modern state management using **Redux Toolkit & RTK Query**, and secure authentication with **JWT tokens**.
